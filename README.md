# TaughtMyHall

**TaughtMyHall** is an educational application based on the **Method of Memory Palace**.  
It supports effective learning through spatial memory and spaced repetition.

Users create memory palaces, rooms, and objects to which flashcards are assigned and reviewed using the SM-2 algorithm.

---

## Features

- Memory palaces
- Furniture as spatial memory anchors
- Flashcards with spaced repetition (SM-2)
- User authentication
- Learning progress tracking
- REST API for mobile clients

---

## Architecture

- **Backend**: Django REST Framework  
- **Database**: PostgreSQL  
- **Frontend**: React Native 

```
Web App → REST API → PostgreSQL
```
